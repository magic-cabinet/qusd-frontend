# QUSD Frontend

Frontend application for QUSD Protocol built with React, TypeScript, and Vite.

## Development

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build

# Run Storybook
bun run storybook
```

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Storybook
